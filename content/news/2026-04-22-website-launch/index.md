We launched the new group website today. It's built with [Hugo](https://gohugo.io) and
[Tailwind CSS](https://tailwindcss.com), deployed on GitHub Pages, and kept
intentionally simple — every post is a markdown file that anyone in the group
can edit.

To add news, open the repository on GitHub, create a new folder under
`content/news/`, and drop an `index.md` file in it. That's it.
